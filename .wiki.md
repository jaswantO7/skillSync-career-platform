# Project Summary
SkillSync is an AI-powered career growth companion designed to help users navigate their professional development. The platform integrates advanced AI features, providing personalized career recommendations, learning roadmaps, and project suggestions. The backend is built with Node.js and MongoDB, while the frontend utilizes Next.js 14, enabling a responsive and interactive user experience.

# Project Module Description
## Core Modules
1. **User Management**: Manages user profiles, authentication via Firebase, and onboarding.
2. **Resume Parsing**: Utilizes AI to extract skills and experiences from resumes.
3. **Career Path Recommendations**: Offers tailored career progression suggestions based on user data.
4. **Learning Roadmap Generation**: Develops structured learning plans to help users achieve their career objectives.
5. **Project Suggestion Engine**: Recommends portfolio projects suited to the user's desired role.
6. **Progress Tracking**: Monitors user progress, gamifies learning, and tracks achievements.
7. **Mentor Chat**: An AI chatbot providing personalized career advice and guidance.
8. **Skill Audit**: Analyzes resumes or LinkedIn profiles to extract skills and provide insights.

# File Description Inventory
- **models/**: Contains Mongoose schemas for User, SkillGraph, CareerPath, Roadmap, Project, and Progress.
- **routes/**: Defines API endpoints for user management, AI services, and progress tracking.
- **services/**: Implements AI integration for resume parsing, career recommendations, and roadmap generation.
- **middleware/**: Includes authentication middleware, error handling, and input validation.
- **server.js**: Initializes the Express server and connects to the MongoDB database.
- **package.json**: Lists project dependencies and scripts.
- **components/**: Contains various UI components used throughout the application.
- **app/**: Contains pages and routing for the frontend application.
- **lib/**: Contains utility functions and API service logic.

# Technology Stack
- **Backend**: Node.js, Express.js
- **Database**: MongoDB with Mongoose ODM
- **AI Integration**: OpenAI GPT-4, LangChain
- **Frontend**: Next.js 14, Tailwind CSS, Framer Motion
- **Authentication**: Firebase Admin SDK
- **File Processing**: Multer, PDF-Parse
- **Validation**: Joi
- **Security**: Helmet, CORS, Rate Limiting

# Usage
1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd skillsync-backend
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Environment Setup**
   ```bash
   cp .env.example .env
   ```
   Fill in your environment variables.

4. **Start MongoDB**
   ```bash
   sudo systemctl start mongod
   ```

5. **Run the application**
   ```bash
   npm run dev  # Development mode
   npm start    # Production mode
   ```

6. **Build the application**
   ```bash
   npm run build
   ```
